ENKRegnskap

Installer på Mac

Regnskap for enkeltpersonforetak – lokalt på din egen maskin.

Installer med én kommando

Åpne Terminal og lim inn denne kommandoen:

curl -fsSL https://kkollsga.github.io/enk-regnskap/install.sh | bash

Laster ned siste versjon, installerer EnkRegnskap.app i Programmer og åpner den. Ingen passord nødvendig.

Hva gjør kommandoen?

  1. Finner riktig versjon for maskinen din (Apple Silicon eller Intel).
  2. Laster ned installasjonsfilen (DMG) fra GitHub.
  3. Kopierer appen til /Applications.
  4. Åpner ENK Regnskap. Du finner den siden i Launchpad og Programmer.

Krever macOS 11 eller nyere.

Foretrekker du å klikke?

Last ned DMG-en manuelt fra siste release (velg arm64 for Apple Silicon, intel for eldre Macer), åpne den og dra EnkRegnskap til Programmer.

Førstegangsåpning: ved manuell nedlasting er appen ikke signert med Apple Developer-ID (beta). Høyreklikk appen → Åpne, eller kjør xattr -dr com.apple.quarantine /Applications/EnkRegnskap.app. Terminal-kommandoen over slipper dette steget.

Personvern

Alle data lagres lokalt under ~/ENK-Regnskap/ på din egen maskin. Ingen skytjeneste, ingen konto.
