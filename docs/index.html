Inntekter, utgifter, vedlegg og foreløpig skatteberegning – lokalt og privat. Ingen konto, ingen abonnement.

Installer med én kommando i Terminal
bash — ENK Regnskap
$ curl -fsSL https://kkollsga.github.io/enk-regnskap/install.sh | bash

Laster ned siste versjon, installerer i Programmer og åpner appen. Ingen passord nødvendig.

🔓 Førstegangsåpning ved manuell nedlasting

Appen er foreløpig ikke signert med Apple Developer-ID (beta). Høyreklikk appen → Åpne første gang, eller fjern karanteneflagget:

xattr -dr com.apple.quarantine /Applications/EnkRegnskap.app

Terminal-kommandoen øverst slipper dette steget – nedlasting via curl setter ikke karanteneflagget.
